Pizza Verace Mozzarella Di Bufala

22 EURO / buc

Pizza Verace Mozzarella di Bufala este produsa in Italia, cu ingrediente autentice, dupa o reteta originara. Blatul este indelung dospit pentru o textura aerata cu pori mari deschisi in stil ciabatta. Centrul pizzei este subtire si crocant in timp ce marginea este inalta si pufoasa. Pizza este coapta cu lemne pe vatra de piatra si este preparata dupa o reteta clasica ce combina perfect aroma sosului cu pulpa de rosii si gustul Mozzarellei din lapte de bivolita cu busuiocul parfumat.

Ingrediente:

  • Făină de grâu
  • Apă
  • Piure de roșii 10.7%
  • Brânză Mozzarella 11.3% (lapte pasteurizat, sare, cheag microbian, cultură starter)
  • Brânză Mozzarella cu lapte de bivoliță 6.3% (lapte de bivoliță pasteurizat, zer (lapte), sare, cheag microbian)
  • Ulei de floarea soarelui
  • Roșii semiuscate 3.8%
  • Pulpă de roșii 1.7%
  • Sare
  • Busuioc 0.8%
  • Ulei de măsline
  • Dextroză
  • Oregano uscat
  • Drojdie
